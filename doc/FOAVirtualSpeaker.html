Class: FOAVirtualSpeaker

FOAVirtualSpeaker(context, options)

new FOAVirtualSpeaker(context, options)

A virtual speaker with ambisonic decoding gain coefficients and HRTF convolution for first-order-ambisonics stream. Note that the subgraph directly connects to context's destination.
Parameters:
Name Type Description
context AudioContext Associated AudioContext.
options Object Options for speaker.
Properties
Name Type Description
coefficients Array Decoding coefficients for (W,Y,Z,X).
IR AudioBuffer Stereo IR buffer for HRTF convolution.
gain Number Post-gain for the speaker.
