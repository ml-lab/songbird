AmbisonicEncoderTable :Float32Array

Pre-computed Spherical Harmonics Coefficients. This function generates an efficient lookup table of SH coefficients. It exploits the way SHs are generated (i.e. Ylm = Nlm * Plm * Em). Since Nlm & Plm coefficients only depend on theta, and Em only depends on phi, we can separate the equation along these lines. Em does not depend on degree, so we only need to compute (2 * l) per azimuth Em total and Nlm * Plm is symmetrical across indexes, so only positive indexes are computed (l * (l + 1) / 2) per elevation.
